# babel-plugin-dynamic-import-polyfill

Add `array.iterator` polyfill for webpack dynamic import

## Installation

```bash
npm install --save-dev babel-plugin-dynamic-import-polyfill
```

## Usage

.babelrc

```json
{
  "plugins": ["babel-plugin-dynamic-import-polyfill"]
}
```
